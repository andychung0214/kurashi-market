# 暮集選物所測試計畫

## 首頁與商品列表

- [ ] 首頁四個分類連結可開啟對應商品列表。
- [ ] 首頁店主選商品皆可開啟詳情。
- [ ] 搜尋「山」至少顯示一筆商品。
- [ ] 切換為「喜帖」後只顯示喜帖。
- [ ] 價格由高至低排序正確。
- [ ] 空搜尋結果提供清除條件動作。
- [ ] 商品分頁保留既有搜尋與篩選條件。

## 後續測試區

### 商品詳情、收藏與購物車

- [ ] 商品詳情顯示分類專屬資訊與正確庫存。
- [ ] 不存在的商品識別碼顯示友善錯誤狀態。
- [ ] 室內設計案例只有預約諮詢動作，不顯示假價格與購買按鈕。
- [ ] 加入購物車後數量標記即時更新，且不可超過庫存。
- [ ] 收藏與最近瀏覽於重新整理後保留。
- [ ] 購物車可以更新數量、移除商品與套用 `KURASHI100`。
- [ ] 空購物車提供返回商品列表的動作。

### 結帳與付款模擬

- [ ] 空購物車無法進入結帳流程。
- [ ] 錯誤表單顯示摘要並將焦點移至第一個錯誤欄位。
- [ ] 信用卡只接受測試 Visa 格式，且訂單不保存完整卡號或安全碼。
- [ ] 信用卡、ATM、超商代碼皆可建立測試訂單。
- [ ] 重複點擊建立訂單時只建立一筆資料。
- [ ] 完成頁與本機帳戶固定顯示測試標示。
- [ ] 重新整理後仍可查看本機測試訂單。

行動裝置及無障礙測試將於最終驗證補齊。
