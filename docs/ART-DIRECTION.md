# 安暮恆生活視覺方向

## 核心風格

「日式侘寂編輯風 × 都會百貨網格」。侘寂負責材質、留白與真實痕跡；百貨網格負責清楚分類、價格與結帳效率。品牌記憶點是直排的季節暖簾標籤與略帶偏移的活版印章。

## 色票

| 名稱 | 色碼 | 用途 |
|---|---|---|
| 米灰紙 | `#EEE9DD` | 全站背景 |
| 深米紙 | `#E3DCCD` | 區塊與頁尾 |
| 墨黑 | `#242720` | 主要文字與深色按鈕 |
| 森林綠 | `#2F5946` | 預設主題 |
| 酒紅色 | `#6E2C3B` | 酒紅主題 |
| 倫敦藍 | `#1F4B64` | 藍色主題 |
| 聰穎黃 | `#8B6710` | 黃色主題，採深色以保持文字對比 |

四組主題需同步改變主色、柔和底色、焦點與選取狀態，禁止只替換按鈕顏色。

## 字體策略

- 展示標題：`Yu Mincho`、`Hiragino Mincho ProN`、`Noto Serif TC`、Georgia、serif。
- 內文：`Avenir Next`、`Yu Gothic`、`PingFang TC`、`Noto Sans TC`、sans-serif。
- 標籤與數字：`IBM Plex Mono`、`SFMono-Regular`、Consolas、monospace。
- 不強制下載網路字型，降低首次載入與隱私成本。

## 元件規格

- 按鈕：直角、至少 44px 高，主要動作使用主題色，次要動作使用描邊。
- 商品卡：4:5 圖片，狀態章略旋轉，名稱、摘要、價格與動作保持固定順序。
- 表單：標籤永遠可見；錯誤文字與欄位建立關聯；不得只靠顏色表示錯誤。
- 導覽：桌機行內呈現，手機使用可展開選單與底部購物車捷徑。
- 空狀態：說明目前狀態並提供唯一下一步，不放無意義插圖。
- 分類入口：使用 `auto-fit` 自動換行，不以固定卡片數量決定版面。

## 動畫原則

- 只在圖片懸停使用 1.025 倍緩慢縮放。
- 不使用循環動畫、彈跳購物車或多處視差。
- `prefers-reduced-motion: reduce` 時停用非必要轉場與平滑捲動。

## 響應式設計

- 手機：375px 為主要驗證寬度，兩欄商品卡、單欄詳情與固定購物車捷徑。
- 平板：768px 開始顯示完整導覽及四欄商品網格。
- 桌機：1200px 以上使用最大 1440px 內容寬度與非對稱雙欄。

## 禁止事項

- 禁止玻璃擬態、霓虹漸層、滿版膠囊按鈕與每張卡片的大圓角。
- 禁止沒有資訊意義的 `01 / 02 / 03` 裝飾編號。
- 禁止把室內設計案例偽裝成可直接購買的固定價商品。
- 禁止過度平滑、失去材質與使用痕跡的商品圖片。
- 禁止在前端顯示或保存任何正式金流憑證。
